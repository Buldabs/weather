# Weather App — сайт погоды с избранным

Сайт для просмотра текущей погоды в любом городе с возможностью сохранять города в избранное.

## Запуск проекта

### 1. Запуск бэкенда (сервер и база данных)

Откройте терминал и выполните:

```bash
cd backend
node server.js
```

### 2. Запуск фронтенда

```bash
cd frontend
npm run dev
```
